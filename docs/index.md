# Spark Privacy Policy ✨

> *Knowledge, like a precious gem, should shine — but never at the cost of privacy.*

Мы уважаем ваши данные. Этот документ определяет, как приложение **Spark** обрабатывает информацию, не прибегая к профилированию личности, навязыванию авторитетов или сбору лишних данных.

## 1. Какие данные мы используем
- Email для OTP-входа (только при регистрации для апгрейда уровней).
- Локальный прогресс прохождения пазлов на устройстве.
- Контекст взаимодействия в момент запроса (без создания постоянного профиля).

## 2. Что мы **не делаем**
- Не определяем и не храним личность пользователей по изображениям, документам или поведению.
- Не создаём психологический или социальный портрет.
- Не передаём данные третьим лицам и не используем аналитику для идентификации личности.

## 3. Принципы взаимодействия с ИИ
- ИИ помогает **ориентироваться**, но не является источником истины.
- Навигация строится по **контексту**, а не по профилю.
- Пользователь может оставаться в версии без обновлений, если так ему спокойнее.

## 4. Безопасность данных
- Поддерживается HTTPS.
- Доступ к репозиторию и сайту защищён стандартами GitHub Pages.
- В приложении используется режим **Sapphire Privacy Mode**, включая затемнение интерфейса для приватности.

## 5. Хранение и удаление данных
- Пазл считается пройденным при решении или нажатии «Показать ответ».
- Прогресс уходит в **Архив опыта**, но не подлежит переигрыванию.
- Email можно удалить по запросу пользователя в приложении (при отзыве аккаунта/прогресса).

## 6. Контакт
Если у вас есть вопросы по обработке данных, вы можете связаться через раздел **Настройки → Профиль → Политика конфиденциальности**.
